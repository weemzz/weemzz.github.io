What information should I give?

When you place an order, besides the type of software, we need information about the type of device you are using, such as the Bit type, brand, and storage type options, as well as the amount of storage capacity.

What types do we provide?

We provide several types, such as a USB drive to install on your device, you can only use it to install, in contrast to the second type, Persistant, where the software is installed on USB, so you only need to plug and turn on your computer to the boot options menu. We also provide CDs, and for ARM type computers, we provide Micro SD cards.

how to install on my computer?

The most important thing in the installation process is backup, just to be careful, unless there is no important data. The next important process is to create Partitions, but this can be done during the install process. See full information here.

Can I request another software?

Sure you can, just give us the name of the software and other info such as USB type, how big storage your choice, then let us do the rest. There must be a Freeware or some paid softwares. Terms and Conditions applied. See full information here.
