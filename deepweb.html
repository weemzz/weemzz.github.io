How to Use
TOR

Download Tor in torproject and follow the steps as instructed. If you use Windows you will get the browser.exe file, double-click it and follow it for installation.
If you are using Linux, after you download you will get a file in the form of a tor-browser-linux64-7.5.6_en-us.tar.xz or another name as the version and architect of your computer, then extract the .xz file, directly from the file manager or through the terminal by typing (for example I save the file in the original directory Download folder) 

tar -xf /Download/tor-browser-linux64-7.5.6_en-us.tar.xz

To start tor browser from terminal, just type this command (remember the file locations may vary) 

cd Download/tor-browser_en-us/browser/start-browser

Or you can double-click on the browser start-icon in the Download/tor-browser_en-us/browser folder, then wait until tor configures it first, after it's done, repeat the steps for the start-browser again, wait until the tor establishing connection , after that the browser will appear. Ok, you can explore the real world of the internet, happy surfing.
If you want browse Deep Web from android, you can download in playstore just search Orbot. just follow instruction. the method to surfing is the same as you use tor on the desktop, see the short tutorial in this section.

How to be Safe
in Deep Web

Many people say how dangerous it is to surf the Deep Web, even some computer damage, or lose a lot of data because of a virus. Yes, that might happen if you visit the Deep Web, but you can also get a virus when you visit the Surface Web with a normal browser like you do now, whether it's on social media, downloading from a site, Flashdisk, or from many things others. So the Deep Web isn't as bad as some people imagine. As long as you follow the instructions and surf politely, you will be fine. following several important points so that you are comfortable surfing the Deep Web :

  • For Linux users, it is strongly recommended to use Tor as an ordinary user, avoid surfing with Root users or Super Users. If you use Windows you should use another User, and first turn off all anti-virus, because it will disturb your surfing.
  • Close all your browser or file manager before you start Tor Browser. although not important, but this is a wise step when you visit a hidden site
  • Look for sites from Duckduckgo and select reddit to find the site address, make sure you read the comments on reddit before opening the site. Avoid Scam sites, or sites without https.
  • Don't download suspicious files, and always turn off your internet connection when opening the downloaded file. Don't open the file with root on Linux, or open the file as Administrator on Windows.
  • Whatever happens, never fill out a form with a payment card number, or anything for payment purposes. Always use a trusted Bitcoin wallet to process transactions, don't use a credit card or paypal.
Most importantly, make sure that Tor and Bitcoin for payments are not prohibited by the government in your country.
