What should You do
After install Linux OS

Linux operating systems have many types, and each distributor has a variety of characteristics and types. But all have the same characteristic outline. Here are some mandatory routine activities for Linux users who use Debian derivatives such as Kali linux, Ubuntu, Parrot SecOS and others.

Make users (very important)

Some applications may not work if you work with Root user on every linux, besides in terms of security which will be very vulnerable, is an unwise behavior that can open a gap for viruses or other cyber crimes. Then you have to make a user use it, so take a shot.
Open terminal on your linux, then type : 
useradd -m weem
Make password for user weem 
passwd weem
It will display : 
Enter new UNIX password:
Retype new UNIX password:
Anything you typing here, is gonna be invisible, so careful to write your password. if you done typing password twice, press enter. Then type : 
usermod -aG sudo weem
this will add user weem have a super power if we call sudo (stand for Super User Do) To see all list of users we can type : 
cat /etc/passwd
Now you have to logout, then login with new user that we have made.

Explanation Command

The useradd is for create user, -m is to create folder for some users. Sometime you will get some form, fullfill with whatever you want to type, just follow, and type Y if you get question Y/N. If you not get that questions, its ok, just continue to make password for user.

passwd is to create password for one or more user. Typing password on terminal for safety is invisible, and the "UNIX" text that appears because Linux is made on the basis of the UNIX kernel, so don't worry about that.

usermod is to give some user a mod that required next command, to see list command for usermod just type usermod -help -a stand for append, to append the user to supplemental groups, is easy way, is to enter user into some group. The G is to not remove user from his/her initial group. So, the -aG can be interpreted "hey dude, give this user into next group, but dont don't remove it from their native group"

sudo is a Super user group, or can be a command if some user want to do something that required super user. If you typing some command like apt-get update or anything else, that go with error that contain "permission Denied" or "Are you Administrator" just type this to solve : sudo !!

or sudo apt-get update then you will asked to give password that you creat earlier. To find out if you have logged in perfectly with the account you just created, open terminal, than you will see weem@distroName:~$ not a root@distroName:~$ anymore.

Mandatory routine activities

The first thing after successfully installing is to open the Terminal, the heart of Linux, usually indicated by a black box icon with a little code image. do the Update and Upgrade by typing 
apt-get update
then type 
apt-get upgrade
you will be given the choice to continue or not, and type the letter y then enter to continue. For some reason, you can skip this option by adding -y at the end of the apt-get command like This 
apt-get -y upgrade
Or you can also use additional "dist" which means the distributor when upgrading to get a stable upgrade from the application that the distributor provides, while third party applications are not included in this process. You should use both. 
apt-get dist-upgrade
or 
apt-get -y dist-upgrade
that's a routine activity that you have to do often, or you can make one command to be 
apt-get update && apt-get -y upgrade && apt-get -y dist-upgrade

Explanation Command

The apt-get mean for apt is stand for Advanced Package Tool, and get is command which need next typing. Update is for updating one or more application that requires updates. Same as the upgrade command which will raise the version of one or more applications.

The -y command is stand for Yes, then you do not need to type the letter Y on the terminal when the question appears after the terminal prepares from the previous command, such as install, update, Upgrade, autoremove or other orders that require approval.

dist-upgrade is command to update the version of the default application from the distribution you are using. While the Upgrade command without the dist prefix is a more global command to raise the application version of all installed systems, including from third parties.

apt-get autoremove is command for removing one or more application that doesn't needed anymore.
Command cd is to visit a folder, for example cd Picture is order to visit folder Picture. You can return to the previous folder with the command cd .. dont forget the space before couple point.

Or you can visiting some folder on a folder by using slash like cd Picture/wallpaper/black then you will directed to the black folder in the wallpaper folder in the Picture folder. If you type cd .. you will directed to wallpaper folder. Or just type cd then you will directed to initial folder. where ever you are, if you type cd you will direct to initial folder.

The ls is for display list on some folder, like ls Picture it gonna display all files and folder on folder Picture. ls Picture/wallpaper/black/ for list of file and folders on black folder.
