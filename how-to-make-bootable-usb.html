tutorial for making bootable usb drive

If there is a How to
Make Bootable USB

Making bootable USB is very easy, and you will be addicted to trying to install other programs because Linux is a very flexible program, so you will find lots of types of Linux that you can use or you are tweaking. Imagine, with this simple tutorial you can make a USB drive into a computer or something super. Tutorial on how to make Bootable USB can be followed by all groups, from experts to children.

let's Start

Making bootable USB
with Application

  1. Plug your USB into a computer, I suggest you use Flash Drive 3.0 even though this isn't necessary, but this will shorten a lot of time, because the next process will take a lot of time. There are a number of flash drive drives that cannot be used to make Bootable USB, make sure you buy or use what can be used for Bootable, usually the general Flash drive is used, except for brands like HP, or the like.
  2. The Device column contains a choice of USB disks that will be formatted and installed, in the example in the Flashdisk image located on H:
  3. "Partition scheme and target system system type" column "File system" column "Cluster size" and the "New volume label" column leave it according to default.
  4. After that, make sure the three options, "Quick format", "Bootable disk using:", and "Create extended label and icon files" are checked. on the Create bootable disk using option: you will find a small disk icon, click the icon then a dialog box will appear in the form of a file manager to direct the ISO file to be used, search for and select the ISO or IMG file that we downloaded earlier.
  5. Wait until the process is complete, DONE will appear on the left side of the bottom corner. Unplug the Flashdisk, and your USB is ready to be used for the next installation process.

Making bootable USB
From Terminal

  1. If you use Linux, I suggest using the terminal to make UBS bootable. First you have to ensure the USB position, this is very important, because this step will format and delete all contents of the intended storage, don't write the USB location incorrectly.
  2. How to see the location of USB you can scan the fdisk -l this is a command to display all storage and full information including the size and partition. Usually if the Flashdisk or MicroSD is located SDB or SDC, in this tutorial my USB location is on /dev/sdc
  3. If your USB has been installed an ISO or IMG file before, then you must format the partition first, first make sure you have unmounted your USB by typing umount /dev/sdc (remember umount, not unmount) then we format the USB pen drive by typing mkfs.vfat /dev/sdc -I this command will format the USB and make the filesystem with the FAT extension.
  4. The next step is typing dd if=/home/WeeM/Download/file.iso of=/dev/sdc
    1. dd is the command to write ISO files in the destination storage.
    2. if stands for Input File, then give the location of the file you saved, in this tutorial I use the file that I saved in the home-Weem-Download folder.
    3. of stands for Output File, then give the location of your pen drive or USB according to what we checked with fdisk -l earlier.
  5. Wait until it's finished, maybe there is no sign of the process, so just wait until it's finished. When finished, remove the flashdisk from the computer, and bootable USB is ready to be used for the next process.
