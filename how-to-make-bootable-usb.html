Making bootable USB
with Application

  1. Plug your USB into a computer, I suggest you use Flash Drive 3.0 even though this isn't necessary, but this will shorten a lot of time, because the next process will take a lot of time. There are a number of flash drive drives that cannot be used to make Bootable USB, make sure you buy or use what can be used for Bootable, usually the general Flash drive is used, except for brands like HP, or the like.
  2. The Device column contains a choice of USB disks that will be formatted and installed, in the example in the Flashdisk image located on H:
  3. "Partition scheme and target system system type" column "File system" column "Cluster size" and the "New volume label" column leave it according to default.
  4. After that, make sure the three options, "Quick format", "Bootable disk using:", and "Create extended label and icon files" are checked. on the Create bootable disk using option: you will find a small disk icon, click the icon then a dialog box will appear in the form of a file manager to direct the ISO file to be used, search for and select the ISO or IMG file that we downloaded earlier.
  5. Wait until the process is complete, DONE will appear on the left side of the bottom corner. Unplug the Flashdisk, and your USB is ready to be used for the next installation process.
