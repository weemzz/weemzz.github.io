Usability of Parrot Sec OS
Tips and Tricks

Parrot Security Operating System, or better known as Parrot Sec OS, is a type of Linux distro that relies on functions for security, almost the same as other types of Linux distributions in the security field, but Parrot Sec OS has the advantage of complete default applications for security, both online or offline. Parrot sec OS is a derivative linux distro from Debian, with the characteristics of the popular apt-get system and easy to use.

What should You do
After install Parrot Sec OS

As with all types of Linux desktops, for Parrot sec Os this uses the apt-get method which is a feature of linux types from the Debian derivative. Please check for important rituals for Linux users on the Required Linux user Activity page. The display uses the Mate desktop environment. It's lightweight and very comfortable to use even though it's not as light as Gnome, but for devices that aren't too old, this type of display is very good. The file size of Parrot sec OS is quite large, reaching around 4GB. Maybe it's because the application and complete program are the convenience of using. Parrot sec OS has set you as a normal user with permission to run the Sudo command, so you no longer need to create a new user when running Parrot. You only need to run it because all the system settings have been set. Except for those of you who want a challenge in tinkering with Linux, you will find freedom in Kali Linux rather than Parrot Sec OS.

How to install Parrot Sec OS
On PC or Laptop

You can install Parrot sec os side by side with the system that is already on your computer, this is called Dual Boot, or you can install it completely for your computer. it depends on your needs. Who is eligible to use this system? security experts use it, or you learn it is also highly recommended to use Parrot because this system is very complete without you having to install other programs. If you use it for daily or just curious, this OS is very good for you. Well, let's start the tutorial on installing Parrot sec os on your computer, some of the things you need are as follows:

  • the first thing you have to prepare is a Flash Drive, at least 8GB, maybe 4Gb is enough, but give more space to store other files if you need them later.
  • Next you need a Parrot sec OS file, this is very important as you know. You can download it according to your computer's architecture, x86 (32-bit) or x64 (64-bit). How do you find out? If you use Windows, right-click on the computer icon (My Computer) select properties, after the dialog box appears, select the System tab, see in the "System Type" column you will find 32bit or 64bit. If you use Linux or MacOS, you can see it from the terminal, type uname -a then enter, all information about your computer will come out, including the architecture.
  • This is a link to download, before you download, you will be given an option according to your wishes or needs. The first option is Security, will download the full features of Parrot sec os. The second option is Home / Workstation, this option will give a lighter download size because what you will get is only normal features, suitable for daily use or computer for office work. And the third option for users of other types of computer devices such as ARM, or the type of Parrot OS for modification, and others. I suggest downloading the first option, this is the download link.
  • If you use Windows, there are several applications to make ISO or IMG bootable USB, such as win32diskimager, Universal-USB-installer, Rufus, UNetbootin. I will use Rufus for this tutorial. If you are using Linux, please click the bootable USB tutorial in this section. MAC users please click this Tutorial.
  • Prepare some time, a piece of patience, a cup of coffee, and Maryam Bread. And let's start creating a Bootable USB drive.

Making bootable USB
with application Program

  • Plug your USB into a computer, I suggest you use Flash Drive 3.0 even though this isn't necessary, but this will shorten a lot of time, because the next process will take a lot of time. There are a number of flash drive drives that cannot be used to make Bootable USB, make sure you buy or use what can be used for Bootable, usually the general Flash drive is used, except for brands like HP, or the like.
  • The Device column contains a choice of USB disks that will be formatted and installed, in the example in the Flashdisk image located on H:
  • "Partition scheme and target system system type" column "File system" column "Cluster size" and the "New volume label" column leave it according to default.
  • After that, make sure the three options, "Quick format", "Bootable disk using:", and "Create extended label and icon files" are checked. on the Create bootable disk using option: you will find a small disk icon, click the icon then a dialog box will appear in the form of a file manager to direct the ISO file to be used, search for and select the ISO or IMG file that we downloaded earlier.
  • Wait until the process is complete, DONE will appear on the left side of the bottom corner. Unplug the Flashdisk, and your USB is ready to be used for the next installation process.

Making bootable USB
From Terminal

  • If you use Linux, I suggest using the terminal to make UBS bootable. First you have to ensure the USB position, this is very important, because this step will format and delete all contents of the intended storage, don't write the USB location incorrectly.
  • How to see the location of USB you can scan the fdisk -l this is a command to display all storage and full information including the size and partition. In this tutorial my USB location is on / dev / sdb
  • If your USB has been installed an ISO or IMG file before, then you must format the partition first, first make sure you have unmounted your USB by typing umount /dev/sdb (remember umount, not unmount) then we format the USB pen drive by typing mkfs.vfat / dev / sdb -I this command will format the USB and make the filesystem with the FAT extension.
  • The next step is typing dd if=/home/WeeM/Download/file.iso of=/dev/sdb
    dd is the command to write ISO files in the destination storage.
    if stands for Input File, then give the location of the file you saved, in this tutorial I use the file that I saved in the home-Weem-Download folder.
    of stands for Output File, then give the location of your pen drive or USB according to what we checked with fdisk -l earlier.
  • Wait until the process is complete, DONE will appear on the left side of the bottom corner. Unplug the Flashdisk, and your USB is ready to be used for the next installation process.
